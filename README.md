# 心灵树洞 (Mind Tree Hole)

这是一个安全的数字空间，在这里，您可以创建多个独立的、受密码保护的“树洞”，每一个都像大树上的一个秘密基地，用来存放您不同的思绪和感受。无论您发送什么，它都会以一个充满同情心的倾听者的身份，作出安慰、聆听和肯定的答复。所有的数据都安全地存储在您自己的浏览器本地，确保了绝对的隐私。

应用内的AI回复和内容分类功能**完全在您的浏览器中运行**，不依赖任何外部API或服务器。这保证了您的对话是即时的、可离线使用的，并且是完全私密的。

## ✨ 核心功能

- **多树洞系统与隐私锁定**: 您可以在一棵“树洞之树”上开启多个独立的聊天空间。每个树洞都可以设置一个独立的4位密码，确保不同心事的私密性。
- **纯本地“AI”伴侣**: 应用内置了一个轻量级的响应逻辑。它能识别问候语，并从一个包含超过100条预设回复的词库中，选择最温暖、支持性的话语来回应您。这不是一个生成式AI，而是一个精心设计的、完全可预测的、旨在提供安慰的倾听者。
- **智能本地分类**: 通过高效的关键词匹配算法，自动将您的消息按情绪（悲伤、高兴、中性）和内容（事件、感情、心情、图片、工作、学习、生活等）进行分类，帮助您后续回顾和理解自己的心路历程。
- **隐私优先**: 所有聊天记录、密码和分类数据 **100% 存储在浏览器的 LocalStorage 中**，不会上传到任何服务器，完全由您掌控。
- **完全离线可用**: 整个应用可以在没有网络连接的情况下正常工作。
- **强大的历史记录管理**: 每个树洞都提供一个专门的历史记录页面，您可以在其中：
  - 按关键词搜索您的消息。
  - 按情绪和内容类型筛选消息。
  - 批量选择并删除记录，以管理您的空间和隐私。
- **支持图片分享**: 您可以发送图片，本地AI会理解图片的存在并作出相应的回应和分类。
- **可视化存储管理**: 应用默认提供 5MB 的本地存储空间。界面上会清晰地显示存储使用情况，当空间不足时会友好地提示（升级功能为概念演示）。

## 🛠️ 技术栈

- **前端框架**: React, TypeScript
- **AI 逻辑**: 纯客户端、基于规则和关键词匹配的逻辑（**无外部API调用**）
- **样式**: Tailwind CSS
- **运行方式**: 无需构建步骤，直接通过现代浏览器支持的 `importmap` 运行。

## 🚀 如何运行与部署

本项目是一个纯粹的、自包含的前端应用，没有复杂的后端或构建过程，可以非常轻松地运行和部署。

### 运行步骤

1.  **获取代码**: 将所有项目文件下载到一个文件夹中。

2.  **启动本地服务器**:
    由于浏览器安全策略的限制，直接用 `file://` 协议打开 `index.html` 可能无法正常工作。您需要一个简单的本地HTTP服务器来托管这些文件。
    在您的项目文件夹根目录下，打开终端并运行以下命令之一：
    - **使用 Node.js `serve`**:
      ```bash
      # 如果您没有安装 serve，请先安装
      # npm install -g serve
      serve .
      ```
    - **使用 Python 3**:
      ```bash
      python -m http.server
      ```
    - **使用 Visual Studio Code**:
      安装 [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) 插件，然后在 `index.html` 文件上右键，选择 `Open with Live Server`。

3.  **访问应用**:
    打开您的浏览器，访问服务器提供的地址（通常是 `http://localhost:3000` 或 `http://localhost:8000`）。

### 部署

由于没有编译步骤，部署过程就是将整个项目文件夹的内容上传到任何静态网站托管服务。

您可以将所有文件部署到例如：
- Vercel
- Netlify
- GitHub Pages
- Firebase Hosting

只需将文件夹内容上传或通过 Git 推送到这些平台，它们就能自动为您部署应用。**无需配置任何API密钥或环境变量。**
